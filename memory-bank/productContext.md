# Product Context

## Purpose
To provide a comprehensive, production-ready stack for building SaaS applications with modern tools.

## Key Features
- **Frontend**: Next.js 15, React 19, Tailwind CSS, Shadcn UI.
- **Backend**: NestJS, ORPC, Drizzle ORM, Better Auth.
- **Editor**: Tiptap-based rich text editor with media support.
- **Infrastructure**: Docker-first, Turborepo.
